  • Public / Private 중 Public 선택

hexo 설치

명령어 npm을 통해 hexo-cli 모듈을 global하게 설치한다는 명령어이다.

블로그 생성

블로그와 관련된 모든 행동은 hexo명령어로 시작된다. 이는 hexo를 설치할 때 -g 옵션을 통해 global로 설치하였기에 가능한 것이다.

hexo init 명령어는 지정된 블로그 명으로 hexo가 알아서 기본적인 블로그를 만들어 준다.
그리고 생성된 블로그 경로로 이동하여 npm install 명령어를 통해 필요한 모듈을 설치한다.
모듈은 package.json에 명시되어 있다.

이렇게 생성된 블로그 폴더 속 내용

포스트 생성하기

  • source/_posts/ 경로 안에 포스팅이 가능한 markdown 파일이 바로 생성된다.
  • 해당 파일 안에 포스팅 내용을 완성했으면, 다음 명령어를 실행하여 변경사항을 빌드하고, 배포한다.

이 두가지 명령어를 동시에 실행할 수도 있다.
